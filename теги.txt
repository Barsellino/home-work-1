<!DOCTYPE>: Визначає тип документа і його версію, щоб браузер знав, як правильно відображати сторінку.

<html>: Визначає початок та кінець HTML документа.

<head>: Містить мета-інформацію про документ, таку як заголовок, мета-теги, стилі і скрипти.

<title>: Встановлює заголовок документа, який відображається на вкладці браузера або в списку вкладок.

<body>: Містить основний вміст документа, такий як тексти, зображення, посилання і т. д.

<h1> до <h6>: Визначають заголовки різних рівнів.

<p>: Визначає абзац тексту.

<a>: Створює посилання.

<img>: Вставляє зображення.

<ul> та <ol>: Визначають ненумерований та нумерований списки відповідно.

<li>: Визначає елемент списку.

<div>: Визначає блок для організації структури документа.

<span>: Визначає контейнер для частини тексту або інших елементів без зміни структури.

<form>: Визначає форму для введення даних.

<input>: Визначає поле для введення даних в межах форми.

<button>: Визначає кнопку.

<label>: Визначає мітку для елемента форми.

<textarea>: Визначає поле для введення багаторядкового тексту.

<select>: Визначає список випадаючих варіантів для вибору зі списку.

<option>: Визначає елемент вибору в межах елемента <select>.

<table>: Визначає таблицю.

<tr>: Визначає рядок таблиці.

<th> та <td>: Визначають заголовок і комірку даних таблиці відповідно.

<thead>, <tbody>, <tfoot>: Визначають групи рядків таблиці (заголовок, тіло, підвал).

<br>: Вставляє розрив рядка.

<hr>: Вставляє горизонтальну лінію.

<strong> та <b>: Визначають напівжирний текст. Обидва теги використовуються для надання семантики напівжирного тексту, але <strong> також надає семантику важливого тексту.

<em> та <i>: Визначають курсивний текст. Аналогічно, <em> додає семантику виділеного тексту, а <i> використовується для зміцнення тексту.

<hr>: Вставляє горизонтальну роздільну лінію.

<pre>: Визначає зону форматованого тексту, зберігаючи пробіли та рядки.

<code>: Визначає фрагмент коду.

<iframe>: Вставляє вміст іншого документа, також може використовуватися для вбудовування відео чи карт.

<audio> та <video>: Визначають відео або аудіо елементи на сторінці.
